# WeInsight 管理后台阶段四后处理与日报 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 自动消费现有 group/article 后处理任务，并交付“手动生成当日临时日报 + 次日 00:10 补偿前一日最终日报”的异步闭环。

**Architecture:** 后处理 Worker 定时调用现有 clean/parse/analyze Service，并从 MySQL 领取日报请求。Web 只创建请求；日报 Repo 使用生命周期元数据 upsert，汇总日报继续从两类日报只读聚合。

**Tech Stack:** Python 3.12、APScheduler 3.10.4、SQLAlchemy、PyMySQL、FastAPI、Jinja2、HTMX、阶段二已固定的 markdown-it-py/Bleach、pytest、现有日报 Service。

## Global Constraints

- 必须先完成阶段三计划。
- 后处理 Worker 不得获取或写入 `wechat_ui_lock`。
- group 和 article process task 必须继续使用独立表和独立失败计数。
- 后处理失败不修改采集 job/run 状态，不停止下一轮采集。
- 分析完成只保持日报 task 为 pending，不立即生成当日未完结报告。
- 管理员手动生成今日报告时状态为 `provisional`，截止时间为请求时间。
- 次日 `00:10` 自动生成前一自然日，状态为 `final`。
- 手动生成历史日期直接为 `final`；不允许生成未来日期。
- 同一 Web 表单重复提交使用 idempotency key 返回同一请求。
- 单个对象失败不能阻止其他对象生成；请求可为 `partial_success`。
- Markdown 禁用原始 HTML并清理 XSS；下载内容只来自已落库脱敏日报。

- [ ] **Step 2: 定义生命周期类型**

`app/domain/report_lifecycle.py`：

```python
class ReportStatus(str, Enum):
    PROVISIONAL = "provisional"
    FINAL = "final"


class GenerationTrigger(str, Enum):
    MANUAL = "manual"
    AUTOMATIC = "automatic"
    COMPENSATION = "compensation"
    LEGACY = "legacy"


@dataclass(frozen=True)
class ReportLifecycle:
    report_status: ReportStatus
    data_cutoff_time: datetime
    generation_trigger: GenerationTrigger
    last_generated_by: str

    @classmethod
    def provisional(cls, cutoff: datetime, generated_by: str) -> "ReportLifecycle":
        return cls(ReportStatus.PROVISIONAL, cutoff, GenerationTrigger.MANUAL, generated_by)

    @classmethod
    def final(cls, cutoff: datetime, trigger: GenerationTrigger, generated_by: str) -> "ReportLifecycle":
        return cls(ReportStatus.FINAL, cutoff, trigger, generated_by)

    @classmethod
    def manual_for_date(cls, report_date: date, now: datetime, generated_by: str) -> "ReportLifecycle":
        if report_date == now.date():
            return cls.provisional(now, generated_by)
        return cls.final(now, GenerationTrigger.MANUAL, generated_by)
```
