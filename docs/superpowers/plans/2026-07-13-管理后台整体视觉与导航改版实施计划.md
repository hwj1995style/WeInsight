# WeInsight 管理后台整体视觉与导航改版 Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 删除登录后的提示页与默认密码提醒，并把整个 Jinja2 管理后台统一改造成方向 A 的专业运维控制台。

**Architecture:** 保留 FastAPI、Jinja2、原生 CSS/JavaScript 和 ECharts。`base.html` 提供统一应用壳，`app.css` 提供设计令牌与共享组件，独立本地脚本负责侧栏折叠和移动抽屉；业务路由、表单字段和数据模型保持不变。

**Tech Stack:** Python 3、FastAPI、Jinja2、原生 CSS、原生 JavaScript、ECharts、pytest、Playwright。

## Global Constraints

- 登录成功和已登录根路径必须进入 `/dashboard`。
- 匿名根路径仍重定向 `/login`。
- 不渲染默认密码提醒，也不改变密码哈希、Session、CSRF 或限流逻辑。
- 桌面侧栏默认展开，可折叠并通过 `localStorage` 持久化。
- 移动端使用抽屉导航，支持遮罩、导航点击和 Escape 关闭。
- 保持全部现有 URL、表单字段、权限、业务数据、ECharts 文本回退和表格横向滚动。
- 禁止外部 CDN、渐变、玻璃拟态、装饰插画和无关新功能。
- 实现必须遵循 TDD；每个任务先看到预期失败，再写最小实现。

- [ ] **Step 4: 实现最小侧栏脚本**

```javascript
(() => {
  const key = 'weinsight.sidebar.v1';
  const shell = document.querySelector('.app-shell');
  const sidebar = document.getElementById('app-sidebar');
  const desktopToggle = document.getElementById('sidebar-toggle');
  const mobileToggle = document.getElementById('mobile-nav-toggle');
  const backdrop = document.getElementById('nav-backdrop');
  if (!shell || !sidebar) return;

  const setCollapsed = (collapsed) => {
    shell.dataset.sidebarState = collapsed ? 'collapsed' : 'expanded';
    desktopToggle?.setAttribute('aria-expanded', String(!collapsed));
    localStorage.setItem(key, collapsed ? 'collapsed' : 'expanded');
  };
  const closeDrawer = () => {
    shell.classList.remove('nav-open');
    mobileToggle?.setAttribute('aria-expanded', 'false');
    if (backdrop) backdrop.hidden = true;
  };
  setCollapsed(localStorage.getItem(key) === 'collapsed');
  desktopToggle?.addEventListener('click', () => setCollapsed(shell.dataset.sidebarState !== 'collapsed'));
  mobileToggle?.addEventListener('click', () => {
    shell.classList.add('nav-open');
    mobileToggle.setAttribute('aria-expanded', 'true');
    if (backdrop) backdrop.hidden = false;
  });
  backdrop?.addEventListener('click', closeDrawer);
  document.addEventListener('keydown', (event) => { if (event.key === 'Escape') closeDrawer(); });
  sidebar.querySelectorAll('a').forEach((link) => link.addEventListener('click', closeDrawer));
})();
```
