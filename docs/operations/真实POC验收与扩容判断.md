# 真实POC验收与扩容判断

本文档用于第八阶段 Task 61。目标是在完成单公众号/订阅号、单核心群和双链路小窗口验证后，判断是否从 1 个扩到 3 个，是否继续观察，或是否回滚。

## 1. 输入材料

验收前必须准备：

```text
Task 58：单公众号/订阅号真实 POC 运行记录
Task 59：单核心群真实轮询 POC 运行记录
Task 60：双链路交错小窗口验证记录
Task 53：日报质量人工评估表
trial-monitor-report 输出
group-runtime-metrics 输出
article-runtime-metrics 输出
```

验收只基于上述记录和指标，不临时扩大账号数量，不临时启动无人值守调度。

## 2. 验收范围

当前验收范围：

```text
1 个实际授权公众号/订阅号
1 个实际授权核心群
1 到 2 小时双链路交错验证
AI 仍保持 dry-run
model_called=0
不注册 Windows 计划任务
```

不允许：

```text
跳过 1 个账号 POC 直接扩到 3 个
跳过 3 个账号 POC 直接扩到最多 20 个
将 AI dry-run 结果作为自动决策
在 No-Go 条件未解除时继续扩容
```

## 3. Go / Watch / No-Go

Go：1 个账号和 1 个核心群都通过，双链路小窗口无核心群超阈值等待。

Go 条件：

```text
单公众号/订阅号真实 POC 结论为 Go
单核心群真实轮询 POC 结论为 Go
双链路小窗口验证结论为 Go
核心群等待未超过阈值
article 链路可中断且可恢复
wechat_ui_lock 未出现长时间占用
日报质量平均分不低于 4.0
AI 仍保持 dry-run 且 model_called=0
```

Watch：存在可定位失败，但可手动恢复。

Watch 条件：

```text
存在少量失败任务，但错误摘要可解释
article 链路失败可回滚到单账号模式
群链路未受明显影响
日报质量平均分低于 4.0 但不低于 3.0
需要继续人工复盘，但无需立即中止
```

No-Go：任一账号连续失败 3 次，或核心群等待超过阈值。

No-Go 条件：

```text
任一账号连续失败 3 次
核心群等待超过阈值
article 链路不可中断
article 链路长时间占用微信 UI
UI 锁超时持续出现
日报质量平均分低于 3.0
AI 不再是 dry-run
CLI 或日志出现聊天正文、文章全文、HTML 或具体文章链接
```

## 4. 扩容判断

扩容：先从 1 个扩到 3 个；3 个稳定后再考虑最多 20 个。

从 1 个扩到 3 个的条件：

```text
Task 58 结论为 Go
Task 59 结论为 Go
Task 60 结论为 Go
日报质量平均分不低于 4.0
连续 1 到 2 小时无核心群超阈值等待
没有账号连续失败 3 次
AI 仍保持 dry-run
```

3 个稳定后再考虑最多 20 个的条件：

```text
3 个实际授权公众号/订阅号人工值守 POC 均通过
任一账号失败都可单独关闭
关闭 article 链路后群链路仍可独立运行
核心群等待未超过阈值
日报质量连续达标
trial-monitor-report 指标可解释
```

不得扩容：

```text
仍处于 Watch 或 No-Go
失败原因不可定位
回滚路径未演练
AI dry-run 未验证 model_called=0
用户可见输出存在敏感内容
```

## 5. 回滚判断

回滚到单账号模式：

```text
article 链路失败集中在某一个账号
任一账号连续失败 3 次
双链路交错验证出现可恢复但需继续观察的问题
日报质量低于扩容阈值
```

只保留群链路：

```text
核心群等待超过阈值
article 链路长时间占用微信 UI
UI 锁超时持续出现
关闭 article 链路后仍需观察群链路稳定性
```

关闭 AI：

```text
AI 不再是 dry-run
model_called 不等于 0
AI 输出被用于自动改写链路状态
```

## 8. 后续动作

Go 后动作：

```text
进入 3 个实际授权公众号/订阅号人工值守 POC
继续保持 AI dry-run
继续每日执行 trial-monitor-report
继续人工复盘日报质量
```

Watch 后动作：

```text
保持 1 个账号和 1 个核心群
修复可定位失败
补充截图、日志和指标证据
再次执行 Task 58、Task 59 或 Task 60
```

No-Go 后动作：

```text
停止扩容
回滚到单账号模式或只保留群链路
复核微信健康、UI 锁、失败任务和日报质量
重新完成真实 POC 前置复核清单
```
