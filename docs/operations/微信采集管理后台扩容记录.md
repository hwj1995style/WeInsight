## 4. 强制停止与回退

出现以下任一情况，立即停止 article 新调度，等待协作检查点，独立确认 UI lock 租约后回退上一批次：

- 重复运行或并发 UI owner；
- 微信连续健康失败或客户端失去响应；
- 核心群等待超过批准阈值；
- Worker 反复重启、租约无法收敛或敏感输出泄漏；
- 次日 00:10 后前一自然日日报未 final。
