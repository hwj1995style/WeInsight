# 汇总日报运行手册

本文档适用于第六阶段汇总日报的人工查看和导出。汇总日报用于把微信群日报和公众号/订阅号日报放到同一份阅读入口，不负责采集、清洗、分析或任务补偿。

## 1. 使用范围

汇总日报只适合在 group/article 各自日报已经生成后执行。

执行前确认：

```text
微信群日报已经按日期生成
公众号/订阅号日报已经按日期生成
开发阶段仍然手动执行命令
不注册 Windows 计划任务
```

## 2. 查看汇总日报

```powershell
$env:WEINSIGHT_MYSQL_PASSWORD='weinsight_dev'
python -m app.main summary-daily-report-show --config config/config.dev.yaml --date 2026-07-06
```

该命令会按指定日期生成汇总日报并输出到控制台。汇总日报不占用微信 UI，不获取 `wechat_ui_lock`。

## 3. 导出汇总日报

```powershell
$env:WEINSIGHT_MYSQL_PASSWORD='weinsight_dev'
python -m app.main summary-daily-report-export --config config/config.dev.yaml --date 2026-07-06 --output runtime/reports/summary
```

默认导出路径：

```text
runtime/reports/summary/YYYY-MM-DD/summary.md
```

`--output` 也可以指向具体 `.md` 文件；此时会直接写入该文件。

## 4. 只读边界

汇总日报只读取 wechat_group_daily_report 和 wechat_article_daily_report。

汇总日报不得执行以下操作：

```text
不得采集微信群消息
不得采集公众号/订阅号文章
不得重跑群分析或文章分析
不得更新 group/article 任务表
不得更新 group/article 游标、采集日志或日报源表
```

汇总日报失败不得回写 group/article 链路状态。失败时只能通过当前命令返回错误信息，后续如需持久化失败记录，也必须使用独立汇总层日志。

## 5. 验收检查

人工验收时检查：

```text
summary-daily-report-show 可输出指定日期汇总内容
summary-daily-report-export 可生成 runtime/reports/summary 下的文件
导出内容包含总览、微信群日报、公众号/订阅号日报和隔离说明
微信群链路任务状态不变化
公众号/订阅号链路任务状态不变化
微信 UI 未被打开或占用
```

回归命令：

```powershell
pytest tests/test_phase_six_plan_docs.py tests/test_sensitive_output_guard.py -q
pytest -q
```
